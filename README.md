# 🚗 Street Wash

Sistema web para agendamento e orçamento de serviços de lavagem automotiva.

## 🚀 Tecnologias utilizadas
- Python
- Flask
- HTML5
- CSS3
- Gunicorn
- Deploy via Render

## 🌍 Acesso online
https://street-wash.onrender.com

## 📌 Funcionalidades
- Página institucional
- Sistema de orçamento
- Interface responsiva
- Integração com WhatsApp
